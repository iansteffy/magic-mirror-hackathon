# Start Hackathon Project on Electron!
Start the application: `node --run start`
For Server Only use: `node --run server`

# Hackathon - Where can insert my modules?

in `modules/`.

`modules/default` are modules that came with the base project

# Hackathon - How to display my modules?

Add them to `/config/config.js`
